## Overview
Gallic War Map is a historical visualization project that maps out the events and locations of Julius Caesar's Gallic Wars.

## Features
- Interactive historical map.
- Key locations with descriptions.
- Timeline of battles.

## Technologies Used
- JavaScript
- Leaflet.js
- GeoJSON
- QGIS
